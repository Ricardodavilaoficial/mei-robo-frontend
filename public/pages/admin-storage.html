ADMIN — Storage Viewer

Lista arquivos via backend (admin-only) e gera Signed URL temporária. Storage Rules ficam intocadas.

Escopo seguro (recomendado): sandbox/...

Como funciona

  1. Selecione UID (ou clique em “Usar meu UID”).
  2. Use o Prefix (por padrão: sandbox/{uid}/contatos/).
  3. Clique em Listar.
  4. Use Abrir para gerar uma Signed URL temporária e abrir em nova aba.

Endpoints: /api/admin/storage/list e /api/admin/storage/signed-url.
