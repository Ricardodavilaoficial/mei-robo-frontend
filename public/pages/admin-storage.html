Página de administração para listar arquivos de um usuário, gerar Signed URL (15min) e validar thumbs/cota.

Como funciona

  1. Selecione UID (ou clique em “Usar meu UID”).
  2. Escolha ano/mês/dia para listar users/{uid}/media/AAAA/MM/DD/.
  3. Clique em Listar arquivos.
  4. Use Abrir para gerar uma Signed URL temporária e abrir em nova aba.

Endpoints tentados: /api/storage/signed-url e fallback para /storage/signed-url.
